# English Learning Tracker

A personal tracker for English classes with Ms. Jessica.

## Live URL

**https://client-lovat-phi-40.vercel.app**

## Features

- 📅 Calendar view to track learning days
- ✅ Mark days as "Learned" or "Off"
- 📝 Add notes for each session
- 📎 Upload attachments (images, PDFs)
- 💾 Data saved in browser LocalStorage

## Tech Stack

- React + TypeScript + Vite
- Deployed on Vercel (free)

## Local Development

```bash
cd client
npm install
npm run dev
```
